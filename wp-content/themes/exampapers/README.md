# Exampapers

Exampapers is a WooCommerce-ready child theme for Twenty Twenty-Five, built for downloadable exam paper stores.

## Notes

- Products should be WooCommerce simple products marked virtual and downloadable.
- Product filtering is prepared for attributes: exam level, exam area, subject, format, difficulty and school.
- Category descriptions should be edited in WooCommerce admin and treated as SEO landing-page copy.
- The theme uses WooCommerce hooks first and keeps template overrides limited to archive, product card, single product content and tabs.
